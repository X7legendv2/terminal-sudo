# Terminal Simulator

Простой симулятор терминала с командами.

## Команды
- `register` - создать аккаунт
- `sudo` - выполнить sudo
- `plasma` - показать случайное число
- `whoami` - показать имя пользователя
- `help` - список команд
- `exit` - выход

## Как запустить
```bash
python terminal.py
